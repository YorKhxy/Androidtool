# Assets

The packaged build ships **no standalone logo / wordmark asset** — the product
identifies itself only by the text **安卓设备监控** (window title, bold white) and a
**Lucide `monitor-smartphone`** glyph in the cornflower-blue accent (`#5597DC`).

The canonical brand lockup is therefore defined in `../preview/brand-lockup.html`
(name set in the brand font + the `monitor-smartphone` mark). The product's full
icon language is **[Lucide](https://lucide.dev)** (`lucide-react@0.290`, confirmed
in `package.json`) — load it from CDN rather than copying files.

If you have an official icon / logo, drop it here and update the README + the
Brand cards.
